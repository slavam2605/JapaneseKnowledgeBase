# Извлечение слов MNN в каталог

Этот скилл — **процесс** «страницы → каталог». Формат файла, место хранения, нумерация и правила
транскрипции — в `japaneseCards/knowledge/mnn-catalog.md` (источник правды, **не дублировать здесь**,
соблюдать строго). Общие соглашения (аудитория, локальный словарь `dict_lookup.py`) — в
`japaneseCards/knowledge/conventions.md`.

## Что делает (и чего НЕ делает)

- **Делает:** читает данные страниц (текст, либо картинки/сканы через визуальный разбор),
  извлекает **все** слова, складывает в справочный каталог `data/reference/mnn/`.
- **НЕ делает:** не создаёт карточки Anki и не пишет в `data/words.md`. Каталог — источник, не
  очередь. Карточки — отдельным явным шагом через скилл `japanese-cards` («сделай карточки из
  reference/mnn/lessonNN.md»).

## Процесс

1. **Определить целевой файл.**
   - Основной урок → `data/reference/mnn/lessonNN.md` (NN с ведущим нулём).
   - Доп. тематический список → `data/reference/mnn/extra/<тема>.md` (имя-тема латиницей).
   - Номер урока / тему взять из слов пользователя или со страницы. Если не ясно — коротко спросить,
     прежде чем писать (один вопрос, не гадать наугад).
2. **Извлечь слова со страниц.** Переписать чтение (かな), кандзи (если книга даёт), русский перевод
   (как в RU-комментарии) и пометку при наличии. Транскрибировать, **не сочинять**: чтения/форму/
   перевод берём со страницы. Нечитаемое (скан/картинка) → `???` в ячейку и проговорить пользователю;
   при сомнении сверить через `tools/dict_lookup.py`, но финальная истина — страница.
3. **Записать в каталог** по формату из `mnn-catalog.md` (шапка-источник + таблица из 4 колонок).
   - Файла нет → создать с шапкой (книга/урок/страницы/дата извлечения).
   - Файл есть → дописать строки в ту же таблицу с **дедупом внутри файла**; в шапке расширить
     диапазон страниц. Папку `data/reference/mnn/` (и `extra/`) создать, если её ещё нет.
4. **Отчитаться:** сколько слов извлечено и в какой файл записано; перечислить флажки `???`/
   сомнительные, если были. Напомнить, что для карточек нужен отдельный запуск `japanese-cards`.

## Чек-лист

- [ ] Целевой файл определён верно (урок с ведущим нулём / `extra/<тема>`).
- [ ] Таблица — ровно 4 колонки `かな｜漢字｜Перевод｜Пометка`, шапка-источник заполнена.
- [ ] かな заполнена во всех строках; 漢字 — только где даёт книга; орфография НЕ нормализована.
- [ ] Дедуп внутри файла сделан; межурочные повторы не трогали.
- [ ] Сомнительные места помечены `???` и названы пользователю.
- [ ] Карточки НЕ создавались (это шаг `japanese-cards`).
