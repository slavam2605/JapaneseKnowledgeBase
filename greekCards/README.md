Этот проект предназначен для обработки карточек Anki для изучения греческого
языка.

# Локальная база карточек

Скрипт `scripts/export_anki_notes.py` читает карточки из Anki через
AnkiConnect и складывает очищенный текст в локальную SQLite-базу
`data/greek_cards.sqlite3`. В Anki он ничего не меняет: разрешены только
read-only действия `version`, `deckNames`, `findNotes` и `notesInfo`.

Экспорт основной колоды для будущих embeddings:

```bash
python3 scripts/export_anki_notes.py --deck "Πες το ελληνικά"
```

Экспорт временной колоды, когда в ней появятся новые карточки:

```bash
python3 scripts/export_anki_notes.py --deck to_process
```

По умолчанию экспорт ограничен типом заметки `Basic Quizlet Extended`. Если
нужно проверить все типы заметок в колоде:

```bash
python3 scripts/export_anki_notes.py --deck to_process --all-note-types
```

Построение локальных embeddings для основной колоды через `BAAI/bge-m3`:

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements-local-embeddings.txt
.venv/bin/python scripts/build_embeddings.py --deck "Πες το ελληνικά" --backend local
```

Перед реальным построением можно проверить объем работы:

```bash
.venv/bin/python scripts/build_embeddings.py --deck "Πες το ελληνικά" --backend local --dry-run
```

Поиск похожих карточек после построения локальных embeddings:

```bash
.venv/bin/python scripts/search_similar.py --backend local --text "βεβαίως"
.venv/bin/python scripts/search_similar.py --backend local --note-id 1713257606856
```

Embeddings сохраняются отдельно для каждого backend/model. Локальная модель
хранится как `local:BAAI/bge-m3`, OpenAI-модель - как
`openai:text-embedding-3-small`, поэтому их можно сравнивать в одной базе.

Построение OpenAI embeddings для основной колоды:

```bash
export OPENAI_API_KEY="..."
python3 scripts/build_embeddings.py --deck "Πες το ελληνικά"
```

Перед реальным вызовом OpenAI тоже можно проверить объем работы:

```bash
python3 scripts/build_embeddings.py --deck "Πες το ελληνικά" --dry-run
```

Поиск похожих карточек после построения OpenAI embeddings:

```bash
python3 scripts/search_similar.py --text "βεβαίως"
python3 scripts/search_similar.py --note-id 1713257606856
```

Если Python на macOS падает с `CERTIFICATE_VERIFY_FAILED`, нужно починить
локальные CA-сертификаты. Скрипты автоматически используют `certifi`, если он
установлен:

```bash
python3 -m pip install certifi
```

Другой вариант - указать CA bundle явно:

```bash
python3 scripts/build_embeddings.py --deck "Πες το ελληνικά" --ca-bundle /path/to/cacert.pem
```

При нулевом балансе OpenAI API embeddings не построятся: после исправления SSL
API вернет ошибку billing/quota. В таком случае можно либо пополнить API-баланс,
либо добавить отдельный локальный backend для embeddings.

# Правила карточек Anki для греческого

Эти правила выведены по примерам из колод `to_process` и `Πες το ελληνικά`.
Колода `to_process` используется как временная зона после импорта Quizlet:
там обычно уже есть греческая сторона, английское объяснение и иногда картинка.
Колода `Πες το ελληνικά` содержит доведенные карточки: к ним добавлен русский
перевод, при необходимости комментарии и блок примеров в формате, который
обрабатывается JS-шаблоном карточки.

## Базовая структура

- Используется тип заметки `Basic Quizlet Extended`.
- Поля заметки:
  - `FrontText` - греческое слово, форма или выражение.
  - `BackText` - перевод, комментарии и блок примеров.
  - `Image` - HTML с картинкой Quizlet, если она есть.
  - `Add Reverse` - включает обратную карточку, обычно `True`.
  - `FrontAudio` и `BackAudio` обычно пустые.
- Теги обычно пустые. Исключение - специальные карточки на формы глаголов:
  `aplos_melodas` для простого будущего и `simple_past` для прошедшего.

## `FrontText`

- На лицевой стороне пишется греческая единица, которую нужно вспомнить.
- Для существительных обычно указывается артикль: `ο`, `η`, `το`, `οι` и т.п.
- Для прилагательных указываются три рода через запятую:
  `επείγων, επείγουσα, επείγον`.
- Для глаголов используется словарная форма настоящего времени:
  `αγνοώ`, `ανθίζω`, `προορίζομαι`.
- Если импорт Quizlet уже содержит несколько нужных форм, их можно оставить в
  `FrontText`: `νυστάζω, νύσταξα, θα νυστάξω`.
- Устойчивые выражения и фразы пишутся целиком:
  `εν πάση περιπτώσει`, `έχω αγωνία`, `λαμβάνω υπόψη`.
- Если нужно показать две равноправные формы, они разделяются HTML-переносами,
  например `ο μηχανικός<br><br>η μηχανικός`.

## `BackText`

- В начале идет значение из Quizlet, чаще всего на английском.
- Русский перевод добавляется отдельной строкой через `<br>`:
  `urgent<br>срочный, неотложный`.
- Если английское значение не нужно или мешает, можно оставить только русский
  перевод.
- Несколько русских вариантов разделяются запятыми.
- Уточнения и предупреждения пишутся в круглых скобках на отдельной строке или
  сразу после перевода:
  `(не σώμα)`, `(не τέλος πάντων)`.
- Пометки вида `(не ...)` автоматически подсвечиваются бледно-красным цветом
  JS-скриптом, если находятся внутри блока с классом `highlight-ne`.
- HTML-переносы используются явно: `<br>` для новой строки и `<br><br>` для
  пустой строки перед блоком примеров.

## Перевод на русский

- Переводить нужно с греческого на естественный русский, а не механически
  пересказывать английское значение из Quizlet.
- Если `FrontText` - одно слово, словарная форма или короткое выражение, русский
  перевод пишется со строчной буквы. Греческие глаголы в форме первого лица
  настоящего времени переводятся русским инфинитивом: `μιλάω` -> `говорить`.
- Короткие устойчивые выражения тоже переводятся как словарная единица, если
  карточка проверяет именно выражение: `έχω αγωνία` -> `волноваться,
  тревожиться`.
- Если `FrontText` - целое предложение или длинная фраза, особенно с заглавной
  буквы, перевод тоже пишется как предложение: с заглавной буквы, с сохранением
  лица, времени и коммуникативного смысла. Не превращать такие фразы в инфинитив
  или словарную пометку.
- Основной русский перевод должен быть кратким и понятным без примеров. Варианты
  значения разделяются запятыми; дополнительные ограничения лучше выносить в
  уточнение или показывать в примерах.

## Примеры

Примеры помещаются прямо в `BackText`, но внутри специальных маркеров:

```html
<br><br><div>[examples]</div>
<div>Греческое предложение с <span style="color: rgb(228, 0, 0);">нужным словом</span>.</div>
<div>Русский перевод с <span style="color: rgb(228, 0, 0);">нужным переводом</span>.</div>
<div><br></div>
<div>Второе греческое предложение.</div>
<div>Второй русский перевод.</div>
<div>[/examples]</div>
```

Правила для блока:

- Блок начинается с `[examples]` и заканчивается `[/examples]`.
- В Anki эти маркеры часто лежат внутри отдельных `<div>`, но JS ищет сами
  строки `[examples]` и `[/examples]`.
- Перед блоком примеров ставится пустая строка: `<br><br>`.
- Каждая строка примера находится в отдельном `<div>`.
- Примеры идут парами: сначала греческое предложение, потом русский перевод.
- Между парами ставится пустой `<div><br></div>`.
- Обычно достаточно 1-3 пар примеров.
- В примерах выделяется изучаемое слово или соответствующий русский перевод:
  `<span style="color: rgb(228, 0, 0);">...</span>`.
- Можно выделять не буквальную форму из `FrontText`, а нужную форму в контексте:
  `αγνοείς` для `αγνοώ`, `ανθίζουν` для `ανθίζω`.
- В русском переводе выделяется смысловой эквивалент, даже если он не дословный.

Правила выбора примеров:

- Примеры должны быть короткими естественными предложениями, а не определениями
  слова и не искусственными фразами ради самого наличия примера.
- Греческое предложение должно показывать обычное употребление изучаемого слова,
  формы или выражения: управление, типичное сочетание, контекст или отличие от
  похожего слова.
- Для отдельного слова или короткого выражения можно использовать естественную
  форму в контексте, даже если она отличается от `FrontText`.
- Для предложения или длинной фразы пример должен сохранять исходную
  конструкцию или близкий к ней речевой шаблон, а не превращать ее в
  словарную форму.
- Русская строка должна быть полноценным переводом всего греческого предложения,
  без пометок языка и без лишних объяснений.

JS-шаблон карточки вырезает содержимое `[examples]...[/examples]` из основного
ответа и переносит его в отдельный блок под заголовком `Примеры:`. Поэтому
основной перевод должен быть понятен и без примеров.

## Обратные карточки

- `Add Reverse` обычно ставится в `True`, чтобы была обратная карточка.
- На обратной карточке JS удаляет блок `[examples]...[/examples]` с лицевой
  стороны, чтобы по русскому/английскому ответу не были видны примеры.
- `Add Reverse` можно оставить пустым для выражений, где обратная проверка
  неудобна или слишком неоднозначна.

## Картинки

- Если Quizlet импортировал картинку, она хранится в `Image` как:
  `<div><img src="quizlet-..."></div>`.
- Картинку не нужно дублировать в `BackText`.
- Поле `Image` может быть пустым.
- В основной карточке картинка показывается после ответа; в обратной карточке -
  на лицевой стороне вместе с переводом.

## Специальные глагольные карточки

- Карточки с тегом `aplos_melodas` показывают заголовок
  `Απλός Μέλλοντας →` на прямой стороне и `→ Απλός Μέλλοντας` на обратной.
- Карточки с тегом `simple_past` показывают заголовок `прошедшее →` на прямой
  стороне и `→ прошедшее` на обратной.
- В таких карточках `FrontText` содержит нужную форму, а `BackText` - базовый
  глагол: например `θα διαβάσω` -> `διαβάζω`, `έκανα` -> `κάνω`.

## Чеклист обработки `to_process`

1. Работать только с карточками из колоды `to_process`.
2. Ничего не менять вне колоды `to_process`.
3. Никуда не перемещать карточки из колоды `to_process`.
4. Карточки с формами глаголов, где с обеих сторон греческий глагол в разных
   временах, не обрабатывать и просто пропускать.
5. Проверить `FrontText`: артикль, родовые формы, ударения, сохранение выражения
   целиком.
6. Проверить английское значение из Quizlet и поправить явные ошибки.
7. Добавить русский перевод отдельной строкой после английского значения, по
   правилам раздела `Перевод на русский`.
8. Добавить уточнение `(не ...)`, если слово легко спутать с другим.
9. Добавить 1-3 пары греческих и русских примеров в блок
   `[examples]...[/examples]`, если карточке нужны контексты, по правилам
   выбора примеров.
10. Выделить изучаемую форму и русский эквивалент красным `span`.
11. `Add Reverse` обычно оставлять как есть. Убирать его только для более
    длинных и сложных греческих фраз, которые неоднозначно переводятся с
    русского обратно на греческий; в сомнительных случаях оставить, владелец
    колоды уберет сам.
12. Не трогать `Image`, если импортированная картинка подходит.
