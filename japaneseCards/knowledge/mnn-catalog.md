# MNN-каталог слов — формат и место (источник правды)

> **STATUS:** единый источник правды по тому, **куда и в каком виде** складывать слова, извлечённые
> из «Минна но Нихонго» (みんなの日本語) и доп. тематических списков. Нужен двум сторонам сразу:
> скиллу `mnn-extract` (пишет каталог) и `japanese-cards` (читает каталог, делает карточки),
> поэтому факт живёт здесь, а скиллы ссылаются — не дублируют (правило из
> [conventions.md](conventions.md)).

## Что это и зачем

Пользователь скармливает страницы RU-комментария MNN (или доп. списки — семья, продукты и т.п.),
а скилл `mnn-extract` извлекает оттуда все слова и складывает в **справочный каталог** по урокам/темам.
Каталог — **стабильный источник** (не очередь): карточки из него делаются **отдельным явным шагом**
(скилл `japanese-cards`, см. ниже). Ничего в Anki из каталога автоматически не уходит.

## Где лежит

Внутри домена, на полке статической справки `data/reference/` (рядом с `existing_grammar_rating.md`):

```
japaneseCards/data/reference/mnn/
  lesson01.md … lesson50.md      # основная лексика по урокам (в MNN 50 уроков, 2 книги × 25)
  extra/
    family.md  food.md  …        # доп. тематические списки (имя файла = тема, латиницей/транслит)
```

- **НЕ** корневой `resources/` репозитория — там бинарные ассеты Kotlin-приложения (JMdict,
  kanjidic2 и пр.); доменной справке карточек там не место.
- **Нумерация урока с ведущим нулём** (`lesson05.md`, не `lesson5.md`) — чтобы сортировались.
- Один урок — один файл. Тематические — в `extra/`, одно имя-тема на файл.

## Формат файла

Маленькая шапка-источник + одна markdown-таблица. Шаблон:

```markdown
# MNN урок 5 (みんなの日本語 II), стр. 12–14

> Источник: みんなの日本語, RU-комментарий. Извлечено 2026-06-29. Транскрипция по книге.

| かな | 漢字 | Перевод | Пометка |
|---|---|---|---|
| きっぷ | 切符 | билет (на транспорт) |  |
| たぶん |  | наверное | нар. |
| かいしゃ | 会社 | фирма, компания |  |
```

Колонки (ровно четыре, в этом порядке):

1. **かな** — чтение слова каной, **как в книге**. Заполнено всегда (это первый/ключевой столбец).
2. **漢字** — запись кандзи, **если книга её даёт**. Если в источнике только кана — оставить пусто.
3. **Перевод** — русский перевод, **как в RU-комментарии** (издание уже на русском). Краткий;
   при многозначности — основное значение, уточнение в скобках (`билет (на транспорт)`).
4. **Пометка** — необязательно. Часть речи (`нар.`, `гл.`, `прил.`), счётный суффикс, краткий
   нюанс. Пусто, если нечего сказать.

Для тематических `extra/*.md` — та же таблица; в шапке `# MNN доп. — семья` вместо номера урока.

## Правила извлечения (против искажений)

- **Транскрибировать, а не сочинять.** Чтения/переводы/форму кандзи берём **из страницы**, не из
  своей памяти. MNN сам — авторитетный источник.
- **Орфографию НЕ нормализуем здесь.** Пишем форму так, как даёт книга. Решение «дать кандзи или
  оставить каной по узусу» — это уже этап **создания карточки** (`japanese-cards`), не каталога.
  Так каталог остаётся точным слепком источника.
- **Фуригана не нужна.** Каталог хранит чтение и кандзи в **разных колонках**, поэтому
  bracket-нотация ` 漢字[かな]` здесь не применяется (она — только в полях карточек, см.
  [conventions.md](conventions.md)).
- **Нечитаемое — флажок, не догадка.** Если страница пришла картинкой/сканом и что-то неоднозначно,
  ставить `???` в ячейку и проговорить это пользователю; при сомнении сверить чтение/значение через
  `tools/dict_lookup.py` (см. «Локальный словарь» в [conventions.md](conventions.md)), но финальная
  истина — страница.
- **Дедуп внутри файла:** один и тот же урок не должен содержать дубль строки. **Межурочные**
  повторы (слово встретилось в двух уроках) — норма, по файлам не чистим; дубли в Anki всё равно
  ловит `canAddNotes` при создании карточек.
- **Дописывание:** если файл урока уже есть, новые строки добавлять в ту же таблицу (с дедупом),
  а в шапке расширить диапазон страниц.

## Связь с созданием карточек

Каталог — **источник**, не очередь. Чтобы сделать карточки из урока, пользователь явно просит, и
`japanese-cards` берёт слова прямо из файла каталога (как уже умеет брать «из data/words.md
строки N–M»). Каталог при этом не меняется; что уже залито в Anki — фиксируется в
[../STATUS.md](../STATUS.md), как и для остальных партий.
